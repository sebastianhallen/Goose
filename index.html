Goose
=====

Run arbitrary commands when saving a file that matches a specified glob.

v1.4.8 - beta
Support for solution reloads.

Note: From this relase there will be two versions of Goose - an experimental/beta/nightly/whenever-something-happens version and a more stable version.
The two versions will have different package ids so if you already have a version of Goose installed, uninstall it if you want to use the non-stable version. If you don't, you will have double installs and both will trigger your actions.
The stable version will be linked as soon as the unstable has been field tested.
Download unstable

To get automatic updates, follow these steps

1. In Visual Studio > Tools > Options  
2. Environment > Extensions and Updates
3. Click "add" and enter the URL for either the stable or latest version.

stable:
http://sebastianhallen.github.com/Goose/version.xml
latest:
http://sebastianhallen.github.com/Goose/version-latest.xml
