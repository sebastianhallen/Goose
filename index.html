Goose
=====

Run arbitrary commands when saving a file that matches a specified glob.

v1.4.8
Support for solution reloads.

Note: From this relase there will be two versions of Goose - an experimental/beta/nightly/whenever-something-happens
version and a more stable version. The two versions will have different package ids so if you already have a version
of Goose installed, uninstall it if you want to use the non-stable version. If you don't, you will have double
installations and both will trigger your actions. The experimental version will only be available via the automatic update feed.

To get automatic updates, follow these steps

1. In Visual Studio > Tools > Options  
2. Environment > Extensions and Updates
3. Click "add" and enter the URL for either the stable or latest version.
